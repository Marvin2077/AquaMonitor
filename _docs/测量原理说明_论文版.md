## 2.1 生物阻抗/电导率测量原理

水体电导率是反映溶解离子浓度的关键水质参数，其物理本质为水溶液在交流电场激励下的导电能力。本系统采用频域阻抗谱方法，基于 ADI 公司 AD5940 模拟前端芯片实现高精度电导率测量。

### 2.1.1 阻抗谱测量方法

**交流阻抗法基本原理**

水溶液电导率测量采用**两端电极交流阻抗法（Two-Electrode AC Impedance Method）**。与直流（DC）测量方法相比，交流激励可有效抑制电极极化效应（electrode polarization）和法拉第反应引起的界面阻抗误差，从而获得更准确的溶液体相阻抗。

设溶液等效电路由体相电阻 $R_s$（反映离子导电能力）与界面双电层电容 $C_{dl}$ 串联构成，则总阻抗的复数形式为：

$$Z(\omega) = R_s + \frac{1}{j\omega C_{dl}}$$

在足够高的激励频率 $\omega$ 下，容抗项 $1/(j\omega C_{dl})$ 趋近于零，阻抗实部近似等于溶液电阻 $R_s$，从而实现对 $R_s$ 的精确提取。[需补充图表：Nyquist 阻抗谱等效电路模型]

**激励频率选择**

本系统默认采用单频激励模式，激励频率设定为 50 kHz。在此频率下，双电层电容效应可忽略，测量结果主要反映溶液体相阻抗。系统同时支持电化学阻抗谱（EIS）扫频模式，扫频范围为 2 kHz～200 kHz，频率点数最多 100 个，可用于分析溶液的频率响应特性。[需补充图表：典型水溶液 EIS Bode 图]

**电导率与阻抗的关系**

溶液电导率 $\sigma$（单位：S/cm）与体相电阻 $R_s$ 的关系由电极常数 $K_{cell}$（单位：cm⁻¹）决定：

$$\sigma = \frac{K_{cell}}{R_s}$$

其中电极常数 $K_{cell} = L / A$，$L$ 为电极间距，$A$ 为有效电极面积，由电极几何形状决定，需通过已知电导率标准液进行实验标定。

### 2.1.2 激励信号生成

**正弦波激励信号的硬件实现**

本系统利用 AD5940 芯片内置的高速 DAC（HSDAC）与片上波形发生器（Waveform Generator）合成正弦激励信号。波形发生器在频率寄存器配置值的控制下，按系统时钟产生离散正弦序列，经 HSDAC 转换为模拟信号。

激励通路如下：

$$\text{波形发生器（数字）} \xrightarrow{\text{HSDAC}} \text{DAC缓冲器} \xrightarrow{\text{ExcitBuf（×2）}} \text{传感器电极}$$

激励电压幅值 $V_{exc}$（峰峰值，单位：mVpp）由 DAC 满幅输出电压与两级增益共同决定：

$$V_{exc} = V_{DAC,FS} \times G_{DAC} \times G_{ExcitBuf}$$

其中 $V_{DAC,FS}$ 为 DAC 满幅参考（内部 1800 mV 参考电压的 80%，约 1440 mVpp），$G_{DAC} \in \{1, 0.2\}$（由 `HsDacGain` 寄存器选择），$G_{ExcitBuf} \in \{2, 0.25\}$（由 `ExcitBufGain` 寄存器选择）。

系统按下式自动选择档位：

$$V_{exc,target} = 1800 \times 0.8 \times \frac{R_{cal}}{R_{TIA}}$$

根据目标激励电压 $V_{exc,target}$ 与预设阈值的比较，自动选取最合适的 $G_{DAC}$ 与 $G_{ExcitBuf}$ 组合，以保证 ADC 输入不过载同时维持足够信噪比。本系统默认激励幅值配置为 **600 mVpp**，激励频率为 **50 kHz**。

**时钟系统**

AD5940 内置高频振荡器（HFOSC），在激励频率 $f_{exc} \geq 20\,\text{kHz}$ 时使能 32 MHz 高性能模式；在 $f_{exc} < 20\,\text{kHz}$ 时切换至 16 MHz 低功耗模式，以在精度与功耗之间取得平衡。

### 2.1.3 信号采集与处理

**电流-电压转换**

传感器电流经高速跨阻放大器（HSTIA）转换为电压信号。HSTIA 的反馈电阻 $R_{TIA}$（本系统默认为 1 kΩ）决定了电流测量量程，其输出电压为：

$$V_{TIA} = I_{sensor} \times R_{TIA}$$

**ADC 滤波链**

模拟电压信号经以下数字滤波链处理后输入 DFT 硬件加速器：

$$\text{ADC（800 kHz/1.6 MHz）} \rightarrow \text{SINC3（OSR=2/4）} \rightarrow \text{SINC2+Notch（OSR=22～1333）} \rightarrow \text{DFT（8192点）}$$

SINC3 滤波器对 ADC 输出进行初次抽取，SINC2 滤波器进一步降低输出数据率并抑制工频干扰（可选 Notch 滤波器用于 50/60 Hz 陷波），最终 DFT 硬件加速器以 8192 点汉宁（Hanning）加窗 DFT 提取基频处的复数分量。

汉宁窗函数 $w(n)$ 的引入可显著抑制频谱泄漏，其定义为：

$$w(n) = 0.5\left(1 - \cos\frac{2\pi n}{N-1}\right), \quad n = 0, 1, \ldots, N-1$$

[需补充图表：ADC 数字滤波链信号流图]

**差分测量结构**

系统分两个时序阶段分别采集电流信号与电压信号：

- **阶段一（电流采集）**：ADC 差分输入接 HSTIA 两端（$V_{HSTIA,P} - V_{HSTIA,N}$），获取 DFT 复数结果 $\hat{I}_{DFT}$（正比于通过传感器的电流）；
- **阶段二（电压采集）**：ADC 差分输入接传感器工作电极（CE0 端，$V_{CE0} - V_{N\_NODE}$），获取 DFT 复数结果 $\hat{V}_{DFT}$（正比于传感器两端电压）。

上述两个阶段由片内序列发生器（Sequencer）按固定时序自动执行，无需 MCU 介入，确保相位一致性。

### 2.1.4 阻抗计算方法

**DFT 原始数据处理**

AD5940 DFT 硬件输出 18 位有符号整数格式的复数实部和虚部，需进行符号扩展后使用：

```
code_signed = code & 0x3FFFF
if (code_signed & (1 << 17)):
    code_signed |= 0xFFFC0000     // 符号扩展至 32 位
```

特别注意：AD5940 DFT 虚部寄存器输出值为实际虚部的**相反数**，电流方向规定（HSTIA_N → HSTIA_P）亦引入一次取反，两次取反相消，最终代码层面对电流复数实部和虚部各取一次反：

$$\hat{I}_{corr} = (-\text{DFT}_{I,real},\ -\text{DFT}_{I,imag})$$

**RTIA 校准与等效阻抗**

在正式测量前，系统需对 HSTIA 反馈电阻进行原位校准（In-situ Calibration）。校准时，开关矩阵将已知精密电阻 $R_{cal}$（默认 1000 Ω）接入测量回路，分别采集 $R_{cal}$ 两端电压 DFT 结果 $\hat{V}_{Rcal}$ 和 HSTIA 两端电压 DFT 结果 $\hat{V}_{RTIA}$，由复数比率法计算 RTIA 的真实复阻抗：

$$\hat{Z}_{TIA} = \frac{\hat{V}_{RTIA}}{\hat{V}_{Rcal}} \times R_{cal}$$

此校准结果存入校准表（$R_{TIA} \in \mathbb{C}$），消除了 RTIA 标称值偏差及寄生电容的影响。

**阻抗计算**

传感器等效阻抗 $\hat{Z}_{sensor}$ 由以下复数运算求得：

$$\hat{I}_{sensor} = \frac{\hat{I}_{corr}}{\hat{Z}_{TIA}}$$

$$\hat{Z}_{sensor} = \frac{\hat{V}_{DFT}}{\hat{I}_{sensor}} = \frac{\hat{V}_{DFT} \times \hat{Z}_{TIA}}{\hat{I}_{corr}}$$

[需补充公式：复数除法展开形式]

**电导率计算**

设 $\hat{Z}_{sensor} = R + jX$，则电导率 $\sigma$（单位：μS/cm）为：

$$G = \frac{R}{R^2 + X^2} \quad \text{（S，溶液电导）}$$

$$\sigma = G \times 10^6 \times K_{cell} \quad \text{（μS/cm）}$$

[需补充图表：完整阻抗计算数据流图]

## 2.2 温度测量原理

水温是影响水体化学反应速率、气体溶解度及电化学传感器响应的重要背景参量，准确的温度测量可为其他参数的温度补偿提供依据。本系统采用铂热电阻（RTD）与高精度 Σ-Δ ADC 的组合方案实现水温测量。

### 2.2.1 温度传感器选型

本系统选用 **PT1000 铂热电阻（Platinum Resistance Thermometer）** 作为温度传感元件。PT1000 符合 IEC 60751 国际标准，其主要技术参数如下：

| 参数 | 数值 |
|------|------|
| 0°C 标称电阻 $R_0$ | 1000 Ω |
| 温度系数 $A$ | $3.9083 \times 10^{-3}\ \text{°C}^{-1}$ |
| 温度系数 $B$ | $-5.775 \times 10^{-7}\ \text{°C}^{-2}$ |
| 温度系数 $C$（$T < 0°C$） | $-4.183 \times 10^{-12}\ \text{°C}^{-4}$ |
| 线性测量范围 | $-200°C \sim +850°C$ |
| 故障保护检测范围 | $100\,\Omega \sim 5000\,\Omega$ |

相较于 PT100，PT1000 具有更高标称电阻，在同等激励电流下可产生更高的测量电压，显著改善系统信噪比，且降低引线电阻对测量精度的影响，适合远端浸入式水质探头应用。ADC 芯片选用 TI **ADS124S08**，该器件为 24 位精密 Σ-Δ ADC，内置可编程增益放大器（PGA）和恒流激励源（IDAC），专为 RTD 测量设计。

---

### 2.2.2 信号调理

**比率法（Ratiometric Measurement）测量架构**

为消除激励电流绝对值误差（IDAC 精度误差、温漂等）对测量结果的影响，本系统采用比率法测量架构。其原理如下：

将精密参比电阻 $R_{ref}$（本系统取 $R_{ref} = 3300\,\Omega$）与 PT1000 串联，IDAC 恒流源（$I_{exc} = 500\,\mu\text{A}$）流经串联回路，以 $R_{ref}$ 两端电压 $V_{ref} = I_{exc} \times R_{ref}$ 作为 ADC 基准电压（REFP - REFN），以 PT1000 两端电压 $V_{PT} = I_{exc} \times R_{PT1000}$ 作为 ADC 差分输入（AIN1 - AIN2）：

$$\frac{V_{PT}}{V_{ref}} = \frac{I_{exc} \times R_{PT1000}}{I_{exc} \times R_{ref}} = \frac{R_{PT1000}}{R_{ref}}$$

此比值与激励电流 $I_{exc}$ 无关，仅取决于两个电阻之比，有效消除了 IDAC 不确定性的影响。[需补充图表：PT1000 比率法测量电路连接图]

**ADS124S08 信号链配置**

| 参数 | 配置值 |
|------|--------|
| 差分输入 | AIN1（PT1000 高端）— AIN2（PT1000 低端） |
| PGA 增益 | 2×（扩大 ADC 满量程利用率） |
| 数据输出速率 | 20 SPS |
| 数字滤波 | 低延迟滤波模式 |
| IDAC 激励电流 | 500 μA（路由至 AIN0） |
| 参考电压 | 外部（$R_{ref}$ 两端，约 1.65 V） |
| SPI 通信 | CPOL=0，CPHA=1（MODE1），4 MHz |

ADC 有效分辨率为 24 bit（有符号，满量程代码 $FS = 2^{23} = 8\,388\,608$），数据就绪信号（DRDY）低电平有效，MCU 以轮询方式检测，超时阈值设为 100 ms（数据周期的 2 倍）。

### 2.2.3 温度计算方法

**第一步：码值转 PT1000 电阻值**

设 ADS124S08 输出 24 位有符号整数 $D_{code}$，则 PT1000 电阻值 $R_{PT}$ 由下式计算：

$$\text{ratio} = \frac{D_{code}}{2^{23}}$$

$$R_{PT} = \frac{\text{ratio} \times R_{ref}}{G_{PGA}} = \frac{D_{code}}{2^{23}} \times \frac{3300}{2} \quad (\Omega)$$

**第二步：电阻值转温度——Callendar-Van Dusen 方程**

IEC 60751 标准规定的 PT1000 电阻—温度关系（Callendar-Van Dusen 方程）为：

- **$T \geq 0°C$**：

$$R(T) = R_0 \left(1 + AT + BT^2\right)$$

整理为标准二次方程：

$$BT^2 + AT + \left(1 - \frac{R}{R_0}\right) = 0$$

令 $Z = 1 - R/R_0$，由求根公式得唯一正实根：

$$T = \frac{-A + \sqrt{A^2 - 4BZ}}{2B}$$

- **$T < 0°C$**：

$$R(T) = R_0 \left[1 + AT + BT^2 + C(T - 100)T^3\right]$$

该方程无解析解，本系统采用 **Newton-Raphson 迭代法**求解，以 $T_0 = -10°C$ 为初始估计，迭代更新：

$$T_{k+1} = T_k - \frac{R(T_k) - R_{measured}}{R'(T_k)}$$

其中导数项：

$$R'(T) = R_0\left[A + 2BT + C(4T^3 - 300T^2)\right]$$

收敛判据为 $|R(T_k) - R_{measured}| < 0.001\,\Omega$，最大迭代次数为 10 次。

[需补充图表：PT1000 R-T 特性曲线及线性近似误差图]

**第三步：多项式校准修正**

由于传感器个体差异、连接电阻及参比电阻精度等因素，原始计算温度 $T_{meas}$ 与真实温度之间存在系统误差。本系统采用**三点二次多项式校准模型**进行修正：

$$T_{true} = a \cdot T_{meas}^2 + b \cdot T_{meas} + c$$

系数 $\{a, b, c\}$ 由三个已知校准点 $\{(T_{meas,i},\ T_{true,i})\}_{i=1}^{3}$ 建立的 $3 \times 3$ 线性方程组确定：

$$\begin{bmatrix} T_{meas,1}^2 & T_{meas,1} & 1 \\ T_{meas,2}^2 & T_{meas,2} & 1 \\ T_{meas,3}^2 & T_{meas,3} & 1 \end{bmatrix} \begin{bmatrix} a \\ b \\ c \end{bmatrix} = \begin{bmatrix} T_{true,1} \\ T_{true,2} \\ T_{true,3} \end{bmatrix}$$

采用高斯消元法（Gaussian Elimination）求解，系数存储于 ESP32 NVS Flash 中，上电后自动加载。校准点默认设定于 25°C、35°C、50°C 三个典型水温档位，每个校准点采集 10 次独立测量取平均值以降低测量噪声。

## 2.3 pH 测量原理

pH 值是水体酸碱性的定量表征，其化学定义为氢离子活度 $a_{H^+}$ 的负常用对数：

$$\text{pH} = -\log_{10}(a_{H^+})$$

本系统采用**安培法电化学传感器**测量工作电极电流，进而通过 Nernst 方程关联 pH 值。

### 2.3.1 电化学测量原理

**Nernst 方程与 pH 响应**

电化学 pH 传感器由工作电极（Working Electrode, WE）、参比电极（Reference Electrode, RE）和对电极（Counter Electrode, CE）构成三电极体系。在恒定偏置电压的控制下，工作电极电流与溶液 pH 值的关系遵循 Nernst 方程：

$$E_{pH} = E_{ref} + S \times (\text{pH}_{ref} - \text{pH})$$

其中：
- $E_{ref}$ 为参比电极电位（V），与测量体系相关；
- $S$ 为 Nernst 斜率，理论值为 $-59.16\,\text{mV/pH}$（25°C）；
- $\text{pH}_{ref}$ 为校准缓冲液的已知 pH 值。

由此可反解：

$$\text{pH} = \text{pH}_{ref} + \frac{E_{ref} - E_{measured}}{S}$$

[需补充图表：三电极体系等效电路图]

**温度对 Nernst 斜率的影响**

Nernst 斜率随温度线性变化：

$$S(T) = -\frac{RT \ln 10}{F} = -\frac{2.303 \cdot R \cdot T}{F}$$

其中 $R = 8.314\,\text{J/(mol·K)}$，$F = 96485\,\text{C/mol}$，$T$ 为绝对温度（K）。在精密测量场合，需以实测温度对 Nernst 斜率进行实时修正（温度补偿）。[需补充公式：$S(T)$ 的数值展开]

### 2.3.2 偏置与激励电路

**低功耗电位控制放大器（LP-PA）**

本系统复用 AD5940 芯片内置的低功耗模拟环路（LP Loop）实现 pH 传感器的电位控制。低功耗 DAC（LPDAC）产生两路直流偏置电压：

- **$V_{bias}$（12位 DAC，接工作电极）**：

$$V_{bias} = \frac{\text{DacData}_{12bit}}{4096} \times V_{ref,2.5V} = \frac{0\text{x}745}{4096} \times 2500\,\text{mV} \approx 1135\,\text{mV}$$

- **$V_{zero}$（6位 DAC，接参比/对电极）**：

$$V_{zero} = \frac{\text{DacData}_{6bit}}{64} \times V_{ref,2.5V} = \frac{0\text{x}17}{64} \times 2500\,\text{mV} \approx 898\,\text{mV}$$

工作电极相对于参比电极的有效偏置电压（Bias Voltage）为：

$$V_{Bias,eff} = V_{bias} - V_{zero} \approx 237\,\text{mV}$$

低功耗电位放大器（LP-PA）维持工作电极电位稳定于 $V_{bias}$，同时驱动对电极以形成闭合电化学回路，实现恒电位仪（Potentiostat）功能。[需补充图表：LP-PA + LPTIA 恒电位仪拓扑图]

**低功耗跨阻放大器（LPTIA）**

工作电极电流 $I_{WE}$ 经低功耗跨阻放大器（LPTIA）转换为电压：

$$V_{LPTIA,out} = V_{zero} - I_{WE} \times R_{TIA,LP}$$

本系统 LPTIA 反馈电阻 $R_{TIA,LP} = 200\,\Omega$，负载电阻 $R_{load} = 100\,\Omega$。在 ADC 满量程约 1.82 V、参考 1.1 V 的条件下，理论可测最大电流约为 ±9 mA。

### 2.3.3 信号采集与处理

**ADC 信号链**

LPTIA 输出信号经以下路径送入 AD5940 内部 ADC：

$$\text{LP-PA/LPTIA 输出} \rightarrow \text{HSTIA\_P 节点} \rightarrow \text{ADC\_P}$$
$$\text{1.1V 内部基准} \rightarrow \text{ADC\_N (VSET1P1)}$$

ADC 差分输入为 $V_{ADC} = V_{HSTIA\_P} - 1.1\,\text{V}$，经 SINC3（OSR=4）→ SINC2（OSR=22）滤波链处理后，直接写入 FIFO（`FIFOSRC_SINC3`，阈值=1）。与电导率测量不同，pH 测量**无需 DFT 处理**，因为测量对象为直流（DC）稳态电流而非交流阻抗。

ADC 配置参数如下：

| 参数 | 配置值 |
|------|--------|
| ADC 时钟 | 1.6 MHz（32 MHz / 20） |
| SINC3 OSR | 4 |
| SINC2 OSR | 22 |
| PGA 增益 | 1× |
| FIFO 源 | SINC3 输出 |
| 每次测量样本数 | 1 |

**原始码值转电流**

设 ADC 输出 16 位有效码值 $D_{code}$，则工作电极电流计算步骤如下：

1. 计算差分码值（减去零点偏移 $D_{zero}$，出厂默认 32768 = ADC 中点）：
$$\Delta D = D_{code} - D_{zero}$$

2. 将差分码值转换为差分电压：
$$\Delta V = \frac{\Delta D}{32768} \times V_{ref,ADC} \approx \frac{\Delta D}{32768} \times 1.82\,\text{V}$$

3. 由电压计算电流（负号源于 LPTIA 的反相特性）：
$$I_{WE} = -\frac{\Delta V}{R_{TIA,LP}}$$

### 2.3.4 pH 换算方法

**基于 Nernst 方程的 pH 计算**

将实测工作电极电位 $E_{measured}$ 代入 Nernst 方程，反解溶液 pH 值：

$$E_{measured} = V_{bias} - I_{WE} \times R_{TIA,LP}$$

$$\text{pH} = \text{pH}_{ref} + \frac{E_{ref} - E_{measured}}{S(T)}$$

其中 Nernst 斜率 $S(T)$ 须以同步测量的水温进行修正（详见 2.2 节）。[需补充公式：温度补偿后的完整 pH 计算表达式]

**两点标定法**

为消除参比电极个体差异和系统偏置误差，采用**两点缓冲液标定法**：

1. **零点标定（Offset Calibration）**：将传感器浸入已知 pH 的标准缓冲液（如 pH = 7.00），记录 ADC 码值 $D_{zero}$，修正零点偏移；
2. **增益标定（Gain Calibration）**：在传感器位置接入已知精度电阻 $R_{ext}$，通过比较理论响应与实测值修正 RTIA 增益系数 $R_{TIA,LP,eff}$。

校准参数（$D_{zero}$，$R_{TIA,LP,eff}$）持久化存储于 ESP32 NVS Flash，上电自动加载。

## 2.4 多通道切换策略

### 2.4.1 传感器通道架构

AquaMonitor 系统同时支持多种水质参数的分时测量，通过两级模拟多路复用器（MUX）实现传感器通道的独立切换，其硬件拓扑如下：

```
传感器阵列
 ├─ 电导率电极  ─── MUX-A S1A/S1B ─┐
 ├─ 余氯电极    ─── MUX-A S2A/S2B  │──→ AD5940 模拟输入
 └─ pH 电极     ─── MUX-A S3A/S3B ─┘

ISFET 传感器阵列（S1～S8）──→ MUX-B ──→ 前端电路 ──→ AD5940/ADC
```

[需补充图表：双 MUX 多通道切换系统框图]

### 2.4.2 MUX-A：传感通道选择器

MUX-A 为**双通道差分多路复用器**（型号待原理图确认，行为特征与 TMUX1134 兼容），采用 2 位 GPIO 地址线控制，支持 3 路有效传感通道（差分 A/B 配对同步切换）：

| 通道 | ADDR1（GPIO18） | ADDR0（GPIO17） | 连接传感器 |
|------|---------------|---------------|-----------|
| CH1 | LOW | LOW | 电导率/阻抗传感器 |
| CH2 | LOW | HIGH | 余氯传感器 |
| CH3 | HIGH | LOW | pH 传感器 |
| CH4（保留）| HIGH | HIGH | 未使用 |

通道切换由 `ChooseSenesingChannel(int channel)` 函数通过 `digitalWrite()` 直接操作 GPIO 完成，切换时延取决于 GPIO 翻转速率（ESP32 典型值约为纳秒量级）。

### 2.4.3 MUX-B：ISFET 通道选择器

MUX-B 为**单端 8:1 多路复用器**（型号待原理图确认），采用 3 位 GPIO 地址线（GPIO25/33/32），支持 8 路 ISFET（离子敏感场效应晶体管）传感器独立寻址，可用于扩展 pH 或其他离子浓度的多点分布式测量。

### 2.4.4 通道切换时序约束

系统状态机保证测量期间通道唯一性：任意时刻仅有一个测量模式处于激活状态（`g_isCondMode` 与 `g_ispHMode` 互斥标志），且 MUX 通道切换**先于** AD5940 初始化与测量启动执行。典型切换时序如下：

```
[上层命令] → ChooseSenesingChannel(N)    // 1. 切换MUX到目标通道
           → AppXXXInit(buf, size)       // 2. 配置AD5940测量模式
           → AppXXXCtrl(START)           // 3. 启动采集
           → AppXXXISR(buf, &count)      // 4. 轮询等待数据就绪
```

## 2.5 系统校准方法

### 2.5.1 电导率电极常数校准

电极常数 $K_{cell}$ 反映电极的几何结构特征，需以已知电导率的标准液（KCl 溶液）进行实验标定。校准流程如下：

1. 将传感器浸入已知电导率 $\sigma_{std}$（如 1413 μS/cm，25°C 标准 KCl 液）的标准溶液；
2. 系统测量传感器原始阻抗 $R_{meas}$；
3. 按下式计算并更新电极常数：

$$K_{cell} = \sigma_{std} \times R_{meas}$$

4. 将 $K_{cell}$ 写入 ESP32 NVS Flash（键名 `"cond_k"`）持久存储。

### 2.5.2 HSTIA RTIA 原位校准

为消除 HSTIA 内部反馈电阻标称值偏差及寄生电容效应对阻抗测量精度的影响，系统在每次初始化时执行 RTIA 原位校准（详见 2.1.4 节）。校准结果以复数形式（$\hat{Z}_{TIA} = R_{TIA,eff} + jX_{TIA,eff}$）表示，存入内存校准表，在每次阻抗计算中作为修正因子使用。

在扫频（EIS）模式下，系统对每个频率点分别独立执行 RTIA 校准，建立包含 100 个频率点的校准复数表（`RtiaCalTable[100]`），以补偿 RTIA 随频率变化的阻抗特性。

### 2.5.3 pH 传感器两点校准

如 2.3.4 节所述，pH 校准采用以下两步流程：

**步骤一：ADC 零点校准（Offset Calibration）**

在 pH = 7.00 标准缓冲液中，记录 ADC 稳态码值作为零点偏移 $D_{zero}$，存储于 NVS Flash（键名 `"ph_off"`）。

**步骤二：RTIA 增益校准（Gain Calibration）**

在传感器接口处接入精密已知电阻 $R_{ext}$，系统测量通过 $R_{ext}$ 的电流，与理论值比对，修正 LPTIA 等效增益电阻 $R_{TIA,LP,eff}$，存储于 NVS Flash（键名 `"ph_rtia"`）。

### 2.5.4 温度三点多项式校准

如 2.2.3 节所述，温度校准采用三点二次多项式拟合，在 25°C、35°C、50°C 三个温度点采集系统测量值与标准计温计真值，构造并求解 $3 \times 3$ 线性方程组，得到修正系数 $\{a, b, c\}$ 存储于 NVS Flash（键名 `"temp_a"`、`"temp_b"`、`"temp_c"`、`"temp_v"`）。每个校准点取 10 次独立测量的算术平均值，以抑制随机噪声。

### 2.5.5 校准数据管理

全部校准系数通过 ESP32 `Preferences` 库（NVS 接口）实现非易失性存储，上电初始化时自动加载。系统支持一键出厂复位（`factory reset` 命令），将全部参数恢复至出厂默认值。
